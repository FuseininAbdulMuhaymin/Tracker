# Finance Tracker

A simple, work-in-progress React app for tracking daily budgets and savings. This prototype shows the UI and navigation I built while learning React and Vite — more features are planned.

## Overview
- Purpose: Personal finance tracker prototype focusing on budgeting and saving flows.
- Completed: Landing page, Daily Budget page, Save page, Tab interface, basic styles and component structure.

## Tech
- Vite + React (JSX)
- Plain CSS for styling

## Run locally
Install dependencies and start the dev server:

```bash
npm install
npm run dev
```

## What I built
- Clean, component-based pages for core user flows.
- Reusable UI pieces and organized assets for easy extension.
- Focus on UX and clear, maintainable code rather than backend features (yet).

## Status & Next steps
- Status: Prototype — core pages and UI are implemented, product is not finished.
- Planned: data persistence (localStorage / backend), charts and analytics, authentication, responsive polish and tests.

If you're a recruiter or reviewer: this project shows focused frontend work and a clear plan to grow into a full product.

Feedback and contributions welcome — I'll keep iterating.
# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
